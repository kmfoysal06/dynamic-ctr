Simple and lightweight plugin for creating and managing custom taxonomies in WordPress.

=== Description ==

The Dynamic Custom Texonomy Register (Dynamic CTR) plugin allows users to easily add and manage custom taxonomies on their WordPress site. It provides a simple UI for creating and configuring custom taxonomies without the need for coding skills.

=== Installation ===
1. Upload the `dynamic-ctr` Folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
2. Visit CPR from Admin Dashboard and add post types via creating new posts.
